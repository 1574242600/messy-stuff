# CopyBiliAccount
复制破站账户  
这玩意基本没用，仅适用于使用大号冲塔的煞笔（比如我）

## 复制内容  
- 关注  
- 追番  
- 收藏  
- 历史记录  
- 番剧观看进度  

## 使用
```bash
npm install
npm run build
npm run start

#按提示输入账户cookies
```
